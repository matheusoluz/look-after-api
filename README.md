# Aplicação para cadastro de produtos - API

## Pré-requisitos

1. Clonar repositório.
2. Instalar Node (>= 10.0.0).
3. Ir para diretório clonado.
4. Instalar dependências.
    > npm install
5. Verificar credênciais de acesso ao CouchDB.
    > user: root <br>
    > password: root
6. Executar API. <br>
    > npm run prod
7. Testar aplicação:
    > 127.0.0.1:3000
